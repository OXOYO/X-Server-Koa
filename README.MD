# X-Server-Koa

## 启动

```
npm install
npm start
```

## API
```bash
api
    \_doSignIn          // post 登录
    \_doSignUp          // post 注册
    \_getUsers          // get 用户列表
    \_getUserById       // get 获取单个用户详情
    \_updateUserById    // post 更新单个用户
    \_removeUserById    // post 删除单个用户
```
